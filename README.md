## Getting started

```
$ composer install
$ yarn
$ cp .env.dist .env
$ touch var/petregister.db
$ bin/console d:d:c
$ bin/console d:s:u --force
$ bin/console server:run
$ yarn run encore dev --watch
```
Open your browser and go to the link provider by server:run (usually http://localhost:8000)
